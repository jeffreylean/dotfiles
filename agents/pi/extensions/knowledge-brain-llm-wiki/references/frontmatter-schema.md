# Frontmatter Schema

Normal vault notes should use:

```yaml
---
title: Human readable title
type: concept # concept | source | project | person | tool | decision | moc | output | inbox | meta
status: seed # seed | active | evergreen | archived
summary: One sentence summary.
aliases: []
tags: []
created: YYYY-MM-DD
updated: YYYY-MM-DD
private: false
llm_include: true
---
```
